첨부하신 현재 **사단법인 소개 페이지**는 “법인 소개”보다 **브랜드형 랜딩**에 더 가깝고, 앞서 확정한 SSoT 기준의 핵심 정보인 **설립 목적, 비전·미션, 주요 활동, 연혁, 후원 CTA**가 약하게 보입니다. 현재 SSoT상 협회 소개 페이지는 최소한 **인사말 / 설립 목적 / 비전과 미션 / 연혁**의 LNB를 가져야 하고, 메인 홈과도 연결되는 수준으로 **협회 소개 요약, 비전·미션, 주요 활동 하이라이트, 최근 활동, 후원 안내**를 구조적으로 보여주는 것이 맞습니다. 또한 현재 문서 기준으로는 **이사장 인사말 전문, 조직도/임원진, 국제교류 구체 사례, 연락처**는 추가 확보 전까지 placeholder로 처리하는 것이 적절합니다.   

또 문강 RIO는 별도의 국가 문화전략 웹진이지만, 협회 본체 페이지에서도 **문강이 어떤 구조의 서브 허브인지**는 연결되어야 합니다. 문강 RIO는 단순 기사형 웹진이 아니라 **질문 구조화 + Reform / Implementation / Outcomes 프레임 + 기사/리포트/원페이저/캘린더/데이터/뉴스레터 배포**를 수행하는 정책형 Media SSoT 서브 허브로 정의되어 있습니다. 따라서 협회 소개 페이지는 법인 정체성을 중심에 두되, 중간 또는 하단에서 문강 RIO와 활동 아카이브로 분기시키는 구조가 맞습니다.

# 1. 현재 페이지의 핵심 문제 진단

첨부 화면을 기준으로 보면 문제는 네 가지입니다.

## 1) 정보 위계보다 스타일이 앞섭니다

현재 상단 Hero는 문구는 강하지만, 실제로 법인 소개 페이지에서 가장 먼저 보여야 할

* 설립 목적
* 비전·미션
* 핵심 활동
* 연혁
  중 무엇이 중요한지 한눈에 들어오지 않습니다.

## 2) 색상 체계가 분산되어 가독성이 떨어집니다

현재 화면은

* 진한 남색 Hero
* 연한 회색 배경
* 다시 진한 그라데이션 카드
* 옅은 회색 카드
  가 섞여 있어 시각적 중심이 흔들립니다. 특히 **진한 카드 안의 흰 텍스트**와 **회색 카드 안의 중간 회색 텍스트**가 번갈아 나와 읽는 피로가 큽니다.

## 3) 카드마다 역할이 불분명합니다

현재 카드 중

* 인사말 카드
* 비전·미션 카드
* 조직 구성 카드
* 주요 연혁 카드
* 참여하는 사람들 카드
  가 병렬로 놓여 있지만, “법인 소개 페이지에서 어떤 순서로 읽어야 하는가”가 명확하지 않습니다.

## 4) 페이지 하단이 중복/과밀해 보입니다

스크린샷상 footer류 블록이 반복되어 보여 전체 완성도가 떨어집니다. 법인 소개 페이지는 오히려 **한 번의 강한 본문 흐름 + 한 번의 정돈된 footer**가 더 적절합니다.

### 2) 진한 배경은 Hero 한 곳만

현재처럼 중간 카드에도 진한 남색 그라데이션을 넣지 말고,

* Hero만 진한 배경
* 나머지 섹션은 밝은 배경 + 흰 카드
  로 통일하는 것이 좋습니다.

### 3) 카드 안 텍스트는 검은색 계열로

중간 회색 텍스트를 많이 쓰면 화면이 탁해집니다.

* 제목: `text-primary`
* 설명: `text-secondary`
* 메타: 더 연한 회색
  으로 정리합니다.

### 4) 본문 규칙

* 본문 기본: `16px / line-height 1.7`
* 카드 본문 최소: `15px`
* Hero 서브카피: `18px / 1.6`
* 긴 문단 폭: `max-width 68ch`

### 5) 카드 스타일

* 그림자 최소화
* border 1px + subtle background
* radius 16~20px
* Hero 아래 첫 카드만 약한 shadow 허용
