# 문화강국네트워크 Media SSoT AI홈페이지

**"문화강국 담론의 공식 정답 본점이자 Media SSoT형 AI홈페이지"**

## 💻 Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Backend / BaaS**: Supabase

## 📁 docs/ 폴더의 역할
이 폴더는 프로젝트 기획 명세서를 코드베이스와 함께 관리하기 위해 이관/내재화된 마크다운 문서 보관소입니다.
모든 개발의 기준점(Single Source of Truth)으로 작동하며, 변경사항 발생 시 이 문서들도 함께 업데이트됩니다.

- `bundle-A-constitution/`: 프로젝트 마스터 브리프, 헌장 및 아젠다 등 기조 문서
- `bundle-B-domain/`: IA 구조, CMS 필드 사전, 카피 및 SEO 기술 명세 등 도메인 문서
- `bundle-C-public-ux/`: UI/UX 디자인 시스템, 컴포넌트, QA 및 런치 플레이북
- `bundle-D-admin-foundation/`: Admin 권한, 데이터 모델링 및 처리 규칙
- `bundle-E-admin-ops/`: Admin Dashboard 화면, 보안 및 Audit Log 정책
- `bundle-F-integration/`: 문서 패키지 최종 정합성 등
- `bundle-G-tech-contract/`: API 엔드포인트 명세, DB 엔티티, 시드/테스트 데이터 스펙
